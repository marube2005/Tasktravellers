## Travelers App
### Overview
- Travelers App is a Flutter-based mobile application that connects passengers (e.g., students, remote travelers) with Matatu Saccos and PSVs in Kenya.
- It simplifies group travel for events, offers affordable on-demand rides from remote areas, and ensures trust via validation.
- No pre-bookingsâ€”post-ride payments via PayHero.
### Key Features
- Passenger Flow: Phone auth (OTP), search rides, create/join group pools, real-time tracking, rate & pay.
- Sacco Flow: Register fleet, manage availability, accept rides, view earnings.
- Validation: Sacco docs upload with admin approval.
- Safety: SOS, in-app chat, emergency contacts.
